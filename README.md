# Hospedagem (hospedagem)

Teste técnico para empresa OnFly.

## Proposta

Desenvolvimento de um SPA Vue capaz de buscar, listar, filtrar e detalhar hotéis.

## Pré-requisitos

NojeJS v22.15.1

## Instalação das dependências

```bash
npm install
```

## Inicializar a API Fake (json-server)

```bash
npm run server
```

## Executar o sistema

```bash
quasar dev
```

## Executar os testes

```bash
npm run test
```
